# Módulo da aplicação Mobile

Este módulo é dedicado apenas a aplicação mobile do JLPTrial que está sob a mesma licença do projeto maior.

Aqui você encontrará todos os assets e códigos responsáveis por gerar o aplicativo tanto em sua versão `android`, quanto (talvez em um futuro próximo), sua versão para `IOS`

Qualquer dúvida que você possa vir a ter, poderá ser consultada verificando a [documentação do repositório principal](https://github.com/JLPTrial/JLPTrial)

## Como compilar este aplicativo

Para simplificar a build, todos os mecanismos tanto para depuração quanto para a versão final, foram implementados no devbox, especificamente para este caso, sugiro que leia o [guia de setup](https://github.com/JLPTrial/JLPTrial/blob/main/docs/SETUP.MD) disponível no repositório principal.
